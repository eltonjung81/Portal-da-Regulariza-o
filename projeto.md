Documentação de Projeto: Sistema "MEI em Dia"1. Visão GeralSistema de automação fiscal para MEIs, focado em regularização de débitos (DAS) e declarações (DASN). O projeto utiliza uma arquitetura de microsserviços simples para garantir segurança e escalabilidade.2. Stack TecnológicaBackend: Python 3.12+ com FastAPI (performance e async).Banco de Dados: PostgreSQL (armazenamento de logs e transações).Automação (Bot/Scraper): Playwright ou Selenium (Python).Processamento de Tarefas: Celery + Redis (para processar a regularização em background sem travar a API).Segurança: Criptografia AES-256 para dados sensíveis em trânsito e em repouso.3. Arquitetura de Endpoints (API)MétodoEndpointDescriçãoGET/api/v1/consultar-cnpj/{cnpj}Consulta dados públicos via BrasilAPI.POST/api/v1/checkoutRegistra a intenção de compra e gera pedido.POST/api/v1/regularizarRecebe as credenciais (pós-pagamento) e enfileira o robô.GET/api/v1/status-processo/{id}Retorna o status da regularização para o usuário.4. O Fluxo de Implementação (Mantra)Fase A: Camada Pública (Landing Page)Use a BrasilAPI para consultar o CNPJ. É gratuita e dispensa web scraping nesta etapa.Exiba na tela: Razão Social, Situação Cadastral e Data de Abertura.Fase B: Segurança e Coleta de DadosNunca armazene a senha do Gov.br permanentemente.Receba a senha via POST seguro (HTTPS).Imediatamente, passe a senha para uma Tarefa de Background (Celery).Após a conclusão da tarefa (geração dos boletos), os dados devem ser limpos da memória.Fase C: O Robô de RegularizaçãoCrie uma classe MEIAutomator utilizando Playwright.Módulo de Login: Automatiza o acesso ao portal do Simples Nacional.Módulo de Consulta: Navega até a área de "Consulta de Débitos".Módulo de Ação: * Plano 59,90: Extrai a lista de débitos e gera a página de download.Plano 119,90: Executa o parcelamento selecionado pelo usuário e emite o DAS da primeira parcela.5. Regras de Segurança (Mandatório)LGPD: Inclua um checkbox obrigatório na página de pagamento: "Autorizo o processamento automatizado dos meus dados fiscais exclusivamente para fins de regularização".Isolamento: Rode o robô em containers Docker separados para cada execução. Se o Gov.br mudar o layout, o contêiner falha apenas para aquele usuário, não derruba o servidor.Próximos passos para você:Configurar o Ambiente: Instale o fastapi e uvicorn.Testar a BrasilAPI: Faça um pequeno script consultar.py para puxar os dados de um CNPJ de teste.Desenhar o Checkout: Integre o Mercado Pago ou Stripe via webhook para que, ao confirmar o pagamento, o seu sistema seja avisado automaticamente e libere o campo da senha para o usuário.
